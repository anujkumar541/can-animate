can-animate-when

  • function
 
  • source

can-animate-when="WHEN"

Parameters

  1. WHEN {String | Object}Optional

    Specify when the animation should occur either as:

    • A string, specifying the name of the event
      • inserted
      • removed
      • click
      • enter
      • submit
    • The name of a property in the template's scope

    If specifying a property, the animation will trigger when the property's value changes. By default, most animations will trigger when the element is inserted.

Use

String

    <div can-animate-when="click" can-animate-fade-out="slow">
        Fade out when clicked
    </div>

Property

    <div can-animate-when="{count}" can-animate-fade-toggle="fast">
        Toggle fade when count changes
    </div>
